# DIV, P, SPAN, A

`div` и `span` — не имеют семантического значения. Они нужны исключительно для стилизации.

Элементы `p`  и `a` имеют семантическое значение. `p` — определяет абзац текста, `а` — обозначает ссылку.

| Tag  | Display | Semantic |
| ---- | ------- | -------- |
| div  | block   | yes      |
| span | inline  | no       |
| p    | block   | yes      |
| a    | inline  | no       |
